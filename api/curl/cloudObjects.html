CloudObject.save(callback)

Saves the CloudObject object to CloudBoost.

Argument
callback
Callback interface object Override the done.
Error Handling

It throws CloudBoostException

var obj = new CB.CloudObject('Student'); //create a new object. obj.set("name", "John Smith"); //store string obj.set("age", 21); // store number obj.set("address", { city: "Hyderabad", state: "Telangana", country: "India" }); // store object
